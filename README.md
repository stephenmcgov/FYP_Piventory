# FYP_Piventory
FYP Stock System

install nodejs:

```https://nodejs.org/en/download/```

clone repo in chosen location via https:

```git clone https://github.com/stephenmcgov/FYP_Piventory.git```

install dependencies to project folder

```cd FYP_Piventory```

```npm install```

start app service

```npm start```
